# Faceme Branding Notes

## Core mark
- Primary mark: uppercase `F`
- Mark color: `#203D5B`
- Canvas color: `#F5F2EA`
- Use the mark with generous spacing and avoid adding effects, gradients, or outlines directly to the monogram

## Identity direction
- Friendly, modern, social, vibrant, premium, clean
- The logo is intentionally minimal, so energy should come from layout rhythm, warm neutrals, and restrained accent color

## Theme tokens
- Brand ink: `#203D5B`
- Brand canvas: `#F5F2EA`
- Accent energy: `#EF6F5E`
- Accent support: `#4D84C4`
- Text base: `#17293D`
- Surface base: `#FFFCF6`

## Usage rules
- Keep the monogram on light warm backgrounds for primary brand moments
- Use `#203D5B` for headers, buttons, and structural emphasis
- Use coral and signal blue as supporting accents, not dominant screen backgrounds
- Avoid purple-heavy, black-heavy, or neon-heavy treatments that would shift Faceme toward clone aesthetics

## Product application
- Login and onboarding use the full lockup
- Feed and top-level discovery surfaces use the compact lockup
- Buttons, avatars, cards, and borders inherit the logo-led neutral and navy palette
